Module 5: Working Through Curriculum

Directions: Follow the steps below and complete the quiz.

It's time to start experiencing things from a student's perspective. Students experience our curriculum using a gitbook and a workbook. The gitbook is equivalent to a textbook and includes prep work which students do before class. It also includes the daily layout for each lesson. The workbook is where students will store their code. Work through the steps below to get started.

1. Fork the work-book and establish your first pull request.

  • Navigate to Austin Coding Academy's Github.
  • Find the name of your class with "-workbook" added to it. (e.g. web-workbook).
  • Fork that repo.
  • Use 'git clone' in the terminal to make a copy of it on your computer.
  • Open the folder in atom, make a change to one of the files, and save it.
  • Use the following git commands to push up your changed to your fork:
    • git add {filename}
    • git commit -m "{explanatory message}"
    • git push origin master
  • Check to make sure your changes pushed to your fork.
  • Then select 'Create pull request' with Austin Coding Academy's version of the repository as your base fork and your version as the head fork.
  • Name your pull request with your first and last name.

2. Review your class textbook. Email for password.

  • Web 101 - Intro to Web
  • FSWD JS 211 - Intermediate JavaScript
  • FSWD JS 311 - Advanced JavaScript
  • FSWD C# 221 - C#
  • FSWD C# 321 - .NET
  • WD 231 - Intermediate Web Design
  • WD 331 - Advanced Web Design
  • WebVR 411 - Virtual Reality Development in the Web Browser

3. Complete all of the projects in the curriculum.

4. Create a lesson plan for day 2 of the class you will be teaching.

5. Complete Module 5 Quiz
